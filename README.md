

#### Random Something

> 中国特色的随机内容生成工具，包括人名，名言，内容，段子，图片，头像，电话，邮箱，地址，学校，网址，新闻等，方便开发者在布局的时候写页面。主要特点是图片和头像可以正常加载图片，运气好的话，还能抽到漂亮小姐姐的图片；段子搞笑好玩，给枯燥的工作加一点乐趣；收录了几乎所有的国内高校，幸运的话，可能下一个出来的就是你的学校。

#### 命令
* Random: 人名
* Random: 名言
* Random: 内容
* Random: 段子
* Random: 图片
* Random: 头像
* Random: 电话
* Random: 邮箱
* Random: 地址
* Random: 学校
* Random: 网址
* Random: 新闻
